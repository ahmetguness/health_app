## Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Expo CLI
- iOS Simulator (for Mac users) or Android Studio (for Android development)

## Installation

1. Clone the repository:
```bash
git clone https://github.com/ahmetguness/health_app.git
cd health_app
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm start
# or
yarn start
```

## Running the App

- For iOS:
```bash
npm run ios
# or
yarn ios
```

- For Android:
```bash
npm run android
# or
yarn android
```

- For web:
```bash
npm run web
# or
yarn web
```
